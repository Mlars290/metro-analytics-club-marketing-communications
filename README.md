# Metro Analytics Club Marketing & Communications Portfolio

## Overview

This repository showcases the marketing, branding, communications, and organizational materials I created while serving as President of the Metro Analytics Club at Metropolitan State University.

I led the club's branding and communication strategy, developed marketing campaigns, created promotional materials, organized speaker events, established the club's Discord community, and built resources to support member engagement and club operations. This portfolio demonstrates how I combined leadership, project coordination, visual communication, and community building to support a growing student organization.

- Created branding assets and visual identity materials
- Developed marketing campaigns to increase student awareness and participation
