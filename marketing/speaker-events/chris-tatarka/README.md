# Chris Tatarka Speaker Event

**Metro Analytics Club Speaker Series | June 8, 2026**

## Event Overview

Hosted a virtual speaker event featuring **Dr. Chris Tatarka**, retired U.S. Army Intelligence Officer, former DOJ Threat Intelligence Analyst, leadership trainer, and Metropolitan State University faculty member.

### Presentation Topic

**Think Like an Analyst: How Professionals Make Decisions When Information Is Incomplete**

The event explored analytical thinking, intelligence methodologies, decision-making under uncertainty, source evaluation, and the role of human judgment in analysis.
